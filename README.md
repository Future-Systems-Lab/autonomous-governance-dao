# Autonomous Governance DAO  

Educational prototype developed for CSU Global coursework in Management Information Systems.  
Implements decentralized leadership and ethical automation using immutable smart contracts.  
Demonstrates data integrity, consent verification, and auditability within digital health frameworks.  

Functional component within the HypnoNeuro and EncryptHealth ecosystem.  
Not deployed to mainnet. For academic and research use only.
